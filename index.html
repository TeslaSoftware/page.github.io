Heuristic Search

Click here for details
Description: Comparisson of 4 different AI algorithms to solve 8-puzzle. Algorithms analyzed are:
- A* with Hamming Heuristics,
- A* with Manhattan Heuristics,
- Iterative deepening A* with Manhattan Heuristic and
- Depth-first Branch and Bound.
All of the algorithms except IDA* checking for duplicates and remove them if their cost is lower. Performance of each algorithm was measured in terms of expanded nodes, number of moves to find the optimal solution and time it took to find optimal solution and finish the algorithm (in nano seconds).

Languages Used: Java
